# Changelog

All notable changes to **solana-actions-blinks-skill** follow [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-06-30

### Added
- Initial release — Solana Actions & Blinks skill for Claude Code / Cursor
- 6 action type modules: tip, SPL, Jupiter swap, NFT mint, governance vote, message sign-in
- Infrastructure skills: spec, blink-builder, actions.json, security, Next.js, testing
- Reference data: constants, SDK API, Jupiter API
- Working templates: transfer-sol, jupiter-swap, actions.json
- 7 commands + 2 agents (actions-architect, blink-engineer)
- CI/CD pipeline: validate, typecheck, install smoke test
- Live devnet demo app (`demo/`) deployable to Vercel
- README banner, tags, usage examples (solana-game-skill style)
